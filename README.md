# Digital Alchemy 

Web zine register for course on `hyperlink.academy`

It was created using [cool template](https://git.sr.ht/~zim/web-zine-01) created by [coolguy](https://coolguy.website)

## Building

- Install racket
- Add pollen package
- Add yaml package


Start the pollen server:

`raco pollen start`
